An Astral Party mod pack which primarily features content from the base game. It migrates the original character skill system using a Relic-based approach and adds chips into the relic pool.

It also includes other unseparated contents, such as: Shape of Dreams, PVZ Fusion Edition, Enigmatic Legacy, Tetre, Mirren: Star Legends, Risk of Rain 2, Community OCs, etc.

All content can be disabled via settings.

[b][u]Important: A content mode toggle button has been added to the top right corner of the main menu:[/u][/b]

[u]It is used to switch between "Vanilla Mode" and "Modpack Mode". The default is Vanilla Mode.[/u]
